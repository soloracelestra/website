📧 Email Setup (EmailJS)

To receive booking emails at soloracelestra@gmail.com, complete the EmailJS setup:

  1. Sign up free at emailjs.com
  2. Add Gmail as an Email Service → copy Service ID
  3. Create an Email Template using the variables listed below → copy Template ID
  4. Go to Account > API Keys → copy your Public Key
  5. Edit the EMAIL_CONFIG object at the top of the <script> section
